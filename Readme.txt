2. 软件说明
======================================================================

    该软件用于同步多个目录中的文件。同步后的目标文件夹将与源文件夹内容
完全一致。同步更新时自动复制源文件夹中大小或日期变更过的文件，并删除目
标文件夹中不存在于源文件夹的文件和目录。

    该软件特别适合经常使用移动硬盘或网络共享目录来备份大量数据的用户。

    如果您在使用中发现任何问题请与 CnPack 开发组联系：

    http://www.cnpack.org
    mailto:master@cnpack.org
